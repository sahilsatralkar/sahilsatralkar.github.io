## ğŸš€ Local Development

To run this website locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sahilsatralkar/sahilsatralkar.github.io.git
   ```



   Open `http://localhost:4000` in your browser
